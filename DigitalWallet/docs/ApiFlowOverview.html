Digital Wallet - Internal API Flow Documentation

A comprehensive map of how each endpoint processes data, which services it invokes, the database it targets, and the resulting action flow.

Auth Service DB: AuthServiceDb

AuthController

Endpoint Controller Action Service Invoked Work Flow & Data State
POST /signup Signup() IAuthService.RegisterAsync() Accepts user data, hashes password using BCrypt, saves to IUserRepository via AuthDbContext. Publishes UserRegistered event to RabbitMQ. Returns token.
POST /login Login() IAuthService.LoginAsync() Validates credentials securely inside AuthDbContext. Calls ITokenService.GenerateTokens() to emit JWT & Refresh token.
POST /refresh-token RefreshToken() IAuthService.RefreshTokenAsync() Validates existing refresh token inside IRefreshTokenRepository, invalidates it, issues a new JWT pair.
POST /logout Logout() IAuthService.LogoutAsync() Revokes the active refresh token from the database.

OTPController & KYCController

Endpoint Controller Action Service Invoked Work Flow & Data State
POST /otp/send SendOTP() IOTPService.SendAsync() Generates 6-digit code, stores it temporarily with expiration in IOTPRepository. Emits event/notifies user.
POST /otp/verify VerifyOTP() IOTPService.VerifyAsync() Matches submitted code against the database. Sets user status to verified.
POST /kyc/submit SubmitKYC() IKYCService.SubmitAsync() Stores ID/document data into IKYCRepository. Sets status to 'Pending' for Admin review.
GET /kyc/status GetStatus() IKYCService.GetStatusAsync() Retrieves the current approval phase of the user's KYC profile.

Wallet Service DB: WalletServiceDb

WalletController (CQRS Pattern)

Endpoint Controller Action Service Invoked Work Flow & Data State
GET /balance GetBalance() IWalletQueryService.GetBalance() Fetches the authenticated user's current absolute balance from WalletServiceDb.
POST /topup TopUp() IWalletCommand.TopUpAsync() Processes a simulated fiat deposit. Commits a TopUp transaction to the Database. Emits TopUpCompleted event via MassTransit.
POST /transfer Transfer() IWalletCommand.TransferAsync() Calculates deductive wallet balances, enforces double-entry rules. Commits to DB context within an atomic transaction. Emits transfer events.
GET /transactions GetTransactions() IWalletQueryService.Get...() Applies pagination and date filters, performs read-only database query returning flattened TransactionDto objects.

Rewards Service DB: RewardsServiceDb

RewardsController

Endpoint Controller Action Service Invoked Work Flow & Data State
GET /account GetAccount() IRewardsService.GetBalance() Retrieves how many loyalty/reward points the user holds. Points are passively generated via Wallet events.
GET /catalog GetCatalog() IRewardsService.GetCatalog() Reads the Rewards catalog (items that can be bought with points) from the DB.
POST /redeem RedeemItem() IRewardsService.RedeemAsync() Takes item ID, verifies sufficient point balance. Deducts points securely, creates a Redemption record, emits RewardRedeemed event.
GET /redemptions GetRedemptions() IRewardsService.GetHistory() Fetches historical data of all items the standard user has redeemed over time.

Notification Service DB: NotificationDb

NotificationsController

Endpoint Controller Action Service Invoked Work Flow & Data State
GET / GetNotifications() INotificationQueryService Returns a paginated list of alerts (e.g., Transfer received, KYC approved) triggered by RabbitMQ events generated across the solution.
GET /templates GetTemplates() ITemplateService Retrieves the HTML/Text templates for emails internally used by the system.
PUT /templates/{id} UpdateTemplate() ITemplateService.Update() Admin or service user updates the format/text of system templates in NotificationDb.

Admin Service DB: AdminServiceDb

KYCController & CampaignController & Dashboard

Endpoint Controller Action Service Invoked Work Flow & Data State
GET /kyc/pending GetPendingKYC() IAdminKYCService Administrators fetch a list of unapproved documents synchronized via distributed messaging or database sync.
POST /kyc/{id}/approve ApproveKYC() IAdminKYCService.Approve() Admin clicks approve. Updates internal DB status, publishes an overarching KYCApproved event to lift user restrictions on the Wallet side.
GET /dashboard GetDashboardStats() IDashboardService Aggregates system-wide data (users count, transfer volume) to display operational metrics to standard Admins.
POST /campaign CreateCampaign() ICampaignService.Create() Inserts a marketing/fidelity campaign into AdminServiceDb. Active campaigns can trigger bonus points in the Rewards loop.

Support Ticket Service DB: SupportServiceDb

UserTickets & AdminTickets Controllers

Endpoint Controller Action Service Invoked Work Flow & Data State
POST /tickets CreateTicket() ISupportService.Create() Normal user generates a Helpdesk ticket. Inserted into SupportServiceDb.
GET /tickets/{id} GetTicketDetails() ISupportService.Get() Retrieves the primary ticket issue and loads all associated threaded replies.
POST /{id}/replies AddReply() ISupportService.AddReply() Adds a threaded comment/reply to an active ticket ID. Emits a generic system-wide Event so Admins/Users get notified.
PATCH /admin/tickets/{id}/status UpdateTicketStatus() IAdminSupport.UpdateStatus() Admin marks a ticket as Resolved, InProgress, etc. Stays in SupportServiceDb.
