👤 Guest

Unauthenticated

Any visitor who has not logged in. Can register or log in to access the system.

🕐 Pending User

Awaiting KYC

Registered user whose KYC documents are pending admin review. Inherits Guest capabilities.

✅ Active User

KYC Approved

Fully verified user with access to wallet, payments, rewards, and support. Inherits Pending User capabilities.

🛡️ Admin

Administrator

Internal staff with elevated privileges for KYC review, user management, support, and analytics.

⚙️ System

Automated

Background automation: RabbitMQ events, wallet auto-creation, reward awards, notifications.
